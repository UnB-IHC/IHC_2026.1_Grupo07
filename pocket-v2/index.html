    Guia de Acessibilidade — Pocket V2

    WCAG 2.2 NBR 17225:2025 e-MAG Atualizado Jun/2026

    Este é o Pocket V2 — um guia prático de acessibilidade digital construído para equipes de desenvolvimento de software usarem no dia a dia: checklists curtos, critérios normativos rastreáveis (WCAG 2.2 / NBR 17225:2025) e recomendações organizadas por papel (dev, design, conteúdo, gestão), não por teoria.

    Como usar este guia

    1. Identifique seu papel no projeto (desenvolvimento, design, conteúdo ou gestão) na navegação abaixo.
    2. Use a página correspondente como checklist durante a implementação — não só na revisão final.
    3. Itens marcados com [NOVO WCAG 2.2] ou [NOVO - TG10/Grupo07] são critérios que não existiam na versão V1 (VerificaAAA, 2024) deste pocket.
    4. Veja o Registro de Mudanças para o histórico completo de evolução da V1 para a V2.

    Desenvolvido como parte do curso de Interação Humano-Computador, ministrado pela docente Rejane Maria da Costa Figueiredo, na Universidade de Brasília (UnB).
