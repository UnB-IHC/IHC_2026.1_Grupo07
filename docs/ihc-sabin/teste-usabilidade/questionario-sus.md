## Cálculo do SUS (uso interno do avaliador — não preencher na frente do participante)

1. Itens **ímpares** (1, 3, 5, 7, 9): some os valores marcados e subtraia 5.
   `(item1 + item3 + item5 + item7 + item9) − 5 = ___`

2. Itens **pares** (2, 4, 6, 8, 10): some os valores marcados e subtraia o resultado de 25.
   `25 − (item2 + item4 + item6 + item8 + item10) = ___`

3. Some os dois resultados e multiplique por 2,5:
   `(resultado 1 + resultado 2) × 2,5 = ___ pontos (0–100)`

**Referência de interpretação:**

| Pontuação | Classificação |
|---|---|
| < 50 | Inaceitável |
| 50–68 | Mediano |
| > 68 | Bom |
| > 80 | Excelente |
